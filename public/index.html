Welcome to SE 2017 JTA Applications

To Apply as a JTA for CSEN 603 - 2017 please follow the steps below:

  1. Submit a POST Request to /api/apply with the following data in the body as a JSON Payload:
    • firstName: String
    • lastName: String
    • email: String
    • mobile: String < Please send as a String to avoid left 0 ommission
    • personalWebsite: String [optional]
  2. If you provided a valid data, a JWT will be created and sent to you in the response body
  3. Attach the JWT in Autherization header and send a PUT Request to /api/applications/confirm with the following data in the body as a JSON Payload:
    • references: [{title: String, url: String}]

      This is an array of objects that refers to your work, you can leave the url blank if it is not required. For example you can include here specific projects, your github profile, or your experience as a JTA in previous years
    • position: Number

      This should be 0 if you are applying for a mentor position (Main JTA with contract) or 1 If you are applying as a volunteer
  4. You should recieve a confirmation response if the request is valid
